# GraphQL Dashboard

A modern, interactive GraphQL Dashboard application for managing user profiles, audit logs, and project data with real-time visualization and analytics.

## Features

- **User Authentication**: Secure login system with tokens
- **User Profile Management**: View detailed user information and login history
- **Audit Logging**: Track and visualize user activities
- **Project Analytics**: Monitor XP progress and project performance with interactive charts
- **Data Visualization**: Beautiful SVG-based charts and progress indicators
- **Responsive Design**: Modern UI built with Tailwind CSS
- **GraphQL Integration**: Seamless API integration with GraphQL endpoints

## Tech Stack

- **Frontend**: HTML5, JavaScript (ES6 Modules)
- **Styling**: Tailwind CSS
- **API**: GraphQL
- **Charts**: SVG-based custom chart implementations

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Sign in with your credentials
4. Access the dashboard to view your profile, audit logs, and project analytics

## Configuration

Edit `config.js` to configure:
- GraphQL API endpoints
- Authentication parameters
- API tokens and credentials

## File Structure

- `index.html` - Login page
- `main.html` - Dashboard page
- `main.js` - Main dashboard logic
- `api.js` - GraphQL API calls
- `auth.js` - Authentication handlers
- `charts.js` - Chart and visualization utilities
- `config.js` - Configuration settings
- `queries.js` - GraphQL queries
